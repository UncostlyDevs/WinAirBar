# Security Policy

## Supported Versions

Security fixes are handled for the latest public WinAirBar source release.

## Reporting A Vulnerability

Report security issues to sag@winairbar.com.

You may also use GitHub issues or the repository security advisory flow when available.

Include:

- A short description of the issue.
- Steps to reproduce it.
- The WinAirBar version or commit you tested.
- Any relevant Windows version details.

Please do not publish exploit details before there has been a reasonable chance to review and fix the issue.

## Security Notes

- WinAirBar does not require administrator privileges.
- WinAirBar stores local settings and history under `%AppData%\WinAirBar`.
- WinAirBar does not collect telemetry.
- Prebuilt EXE releases should be distributed with a SHA256 checksum.
